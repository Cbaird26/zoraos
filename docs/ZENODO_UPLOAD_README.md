**Description:**
```
This paper presents ZoraASI, the governed operating profile of ZoraOS — a complete, local-first, model-agnostic operating system for persistent AI agents. Unlike chatbot interfaces that treat models as disposable endpoints, ZoraASI treats the operating layer as the enduring asset: durable orchestration, user-owned memory, governed tool execution, audit provenance, and multi-machine topology form the substrate upon which any model can operate safely.

The system implements a bounded tool-use loop with per-task budgets, a default-deny governance policy with capability classification, an SHA-256 hash-chained audit ledger, and a full REST API for task submission, cancellation, monitoring, and audit retrieval. A multi-machine deployment (M5 orchestrator, M4 inference, M2 automation, M1 memory) separates concerns across physical nodes.

35 automated tests verify governance enforcement, budget limits, and audit chain integrity. The companion dissertation (Zenodo record 21313827) provides the grounding corpus.
```
